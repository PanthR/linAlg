Module Matrix

Author(s): Haris Skiadas, Barb Wahl

Javascript implementation of Linear Algebra Matrices.

Matrix(arr, options)

Options may be:

  1. A number of rows (assumes row-wise)
  2. An object with possible keys: nrow, ncol, byRow (boolean)

Matrix#toIndex(i, j)

'this' is set to the matrix Returns the vector index that would correspond to the i-th row and j-th column
